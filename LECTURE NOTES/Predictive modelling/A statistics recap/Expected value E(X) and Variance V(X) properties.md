let $X$ and $Y$ be two random variables, $k$ a real number and $K$ a constant random variable such that $P(K = k) = 1$

## EXPECTED VALUE PROPERTIES

> $E(X + Y) = E(X) + E(Y)$
> $E(K X) = k E(X)$
> $E(K) = k$
> $E(X + K) = E(X) + k$

if $X$ and $Y$ are independent then:
> $E(XY) = E(X) E(Y)$


## VARIANCE PROPERTIES

> $V(X) = E(X^2) - E^2(X)$
> $V(K X) = k^2 V(X)$
> $V(K) = 0$
> $V(X + K) = V(X)$

if $X$ and $Y$ are independent then:
> $V(X + Y) = V(X) + V(Y)$
